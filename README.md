# Portfolio — Manav Kheni

Personal portfolio website built with Next.js 14, Tailwind CSS, and Framer Motion.

🔥 **Live:** [portfolio-manav-kheni.vercel.app](https://portfolio-manav-kheni.vercel.app)

## Tech Stack

- **Framework:** Next.js 14 (App Router)
- **Styling:** Tailwind CSS v4
- **Animations:** Framer Motion
- **Fonts:** Inter + JetBrains Mono
- **Deployment:** Vercel
- **Contact Form:** Formspree

## Features

- Ambient ember particle canvas background
- Ember trail cursor with spark physics
- Keyboard burn interaction — type anything, watch it ignite
- 3D flip project cards with live demo links
- Ambient fire sound toggle via Web Audio API
- Scroll-triggered animations throughout
- Fully responsive

## Run Locally

```bash
git clone https://github.com/manavkheni1/portfolio.git
cd portfolio
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
